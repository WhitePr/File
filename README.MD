# 如何安装

本项目使用了以下库
* Gtkmm4.0
* Imagemagick

以及您的C++编译器需要最低支持C++17

### Arch Linux

```sh 
pacman -S gtkmm-4.0 imagemagick
```
### 构建

本项目使用cmake
```sh
mkdir builid
cd build
cmake .. && cmake --build . 
```
您可以在bin文件夹找到编译好的可执行文件
